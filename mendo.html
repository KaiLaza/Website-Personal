Pengenalan JavaScript, CSS, dan HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web.

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan gaya halaman web yang ditulis dalam HTML.

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis.

Contoh Elemen HTML:

<h1>Ini adalah judul</h1> <p>Ini adalah paragraf.</p> <img src="gambar.jpg" alt="Gambar"> <a href="https://www.Google.com">Tautan</a>

Contoh Kode CSS:

body { background-color: lightblue; font-family: Arial, sans-serif; } h1 { color: navy; } p { color: darkgreen; font-size: 18px; }

Contoh Skrip JavaScript:

<script> function sapa() { var nama = prompt('Masukkan nama Anda:'); alert('Halo, ' + nama + '! Selamat datang di halaman kami.'); } </script>
